## The problem we were solving

When Stellar launched smart contracts, developers faced a familiar problem: too much boilerplate, not enough tooling. We'd seen this pattern before with other ecosystems, and we knew what it would take to fix it.

Scaffold Stellar was born out of our own frustration building production apps on the network.

## What Scaffold Stellar is

Scaffold Stellar is an open-source CLI framework that generates production-ready project templates for Stellar smart contract development. Think `create-next-app` but for Stellar dApps.

It includes:
- Project scaffolding with TypeScript support
- Pre-configured contract templates
- Testing infrastructure out of the box
- Deployment scripts for testnet and mainnet
- Frontend integration helpers

## Why it matters

Before Scaffold Stellar, getting a working Stellar smart contract app from zero to deployed took days. With the framework, it takes minutes.

We've seen teams cut their initial setup time by 90%, which means more time building actual features and less time fighting configuration.

## Open source from day one

We built Scaffold Stellar in the open because we believe the ecosystem rises with the tide. Better developer tooling means more builders, which means more value for everyone.

The code is on GitHub. Contributions welcome.
