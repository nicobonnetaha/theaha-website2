## Overview

Scaffold Stellar is the go-to starting point for developers building on Stellar's Soroban smart contract platform. It packages everything a team needs to go from zero to a working dApp in minutes, not days.

We built it in close collaboration with the Stellar Development Foundation, who needed a reference implementation that would lower the barrier to entry for the new wave of Soroban developers.

## The problem

When Soroban launched, developers faced a steep bootstrapping curve. Setting up a project meant manually wiring together the Stellar SDK, a frontend framework, wallet integration, contract deployment scripts, and a local testing environment, all from scratch, every time.

The fragmented tooling landscape meant teams were spending days on project scaffolding instead of shipping product.

## What we built

Scaffold Stellar provides a batteries-included project template that includes:

- A production-ready Soroban contract boilerplate with Rust tooling configured
- A Next.js frontend pre-wired to Stellar Wallets Kit for seamless wallet connection
- Freighter and other Stellar wallet adapters out of the box
- Local development environment with Docker-based Soroban network
- End-to-end deployment scripts for testnet and mainnet
- TypeScript bindings auto-generated from contract interfaces

## Developer experience first

The design philosophy behind Scaffold Stellar was to eliminate every unnecessary decision a developer has to make when starting a project. File structure, build tooling, wallet connection, all of it is pre-decided with sensible defaults, while remaining fully customizable.

We iterated on the DX with real developers, running workshops and gathering feedback to ensure the template reflected how teams actually work.

## Impact

Since launch, Scaffold Stellar has become the standard starting point for Soroban development:

- Used by hundreds of development teams worldwide
- Referenced in the official Stellar documentation
- Adopted by ecosystem grant recipients as their default project setup
- Featured in Stellar developer workshops and hackathons

## Open source

Scaffold Stellar is fully open source under the MIT license. We believe developer tooling should be a public good for the ecosystem, available to every builder, regardless of budget or team size.
